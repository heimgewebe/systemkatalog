# Sources of Truth

Bestand ist ein datierter Index und keine primäre fachliche Wahrheit.

- Code und Historie: zuständiges Git-Repository
- Contracts und Schemas: zuständiges Contract-Repository
- CI: konkreter Workflow-Lauf und Commit
- Runtime: überprüfbare Dienst- und Systemausgaben
- Repositoryrolle: commit-verankerte Repository Reference
- Cabinet-Karten: Navigation, Zustandssnapshot und Provenienz

Bei Konflikten gilt die aktuellere primäre Quelle. Veraltete Snapshots bleiben sichtbar als veraltet markiert.
