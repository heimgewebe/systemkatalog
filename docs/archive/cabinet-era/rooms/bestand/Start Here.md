# Start Here

Der Bestand ist das Inventar von Cabinet.

Hier liegen Repositorykarten, Projektkarten und Quellenregister. Jede Aussage nennt Quelle, Zeitpunkt und – soweit vorhanden – Commit oder Lauf-ID.

## Ablauf

1. Quelle erfassen.
2. Repository oder Projekt zuordnen.
3. Frische und Provenienz dokumentieren.
4. Unklare oder widersprüchliche Angaben an Prüfung übergeben.

Neue Entscheidungen und Aufgaben gehören in Steuerung.
