# Start Here

Steuerung ist der Einstieg für offene Arbeit.

1. Lage und bestätigte Befunde lesen.
2. Entscheidung oder Priorität festhalten.
3. Aufgabe mit Scope, Risiko, Stop-Kriterium und Target-Proof formulieren.
4. Übergabe an einen getrennten Diagnose- oder Coding-Agenten vorbereiten.
5. Ergebnis nach der Umsetzung erneut prüfen.

Keine automatische Ausführung.
